## 🐍 Snake Animation Setup (If you want to add it)

Create `.github/workflows/snake.yml` in your repository:

```yaml
name: Generate snake animation

on:
  schedule:
    - cron: "0 */12 * * *" 
  workflow_dispatch:
  push:
    branches:
    - master

jobs:
  generate:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    
    steps:
      - name: generate snake.svg
        uses: Platane/snk/svg-only@v2
        with:
          github_user_name: madhavpathak
          outputs: dist/snake.svg

      - name: push snake.svg to the output branch
        uses: crazy-max/ghaction-github-pages@v2.6.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Then add this line where you want the snake to appear:
```markdown
![Snake animation](https://raw.githubusercontent.com/madhavpathak/madhavpathak/output/snake.svg)
```
